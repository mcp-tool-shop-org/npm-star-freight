Star Freight is a text-first tactics merchant RPG about hauling freight through a politically fractured star system. This handbook covers the npm launcher package -- the zero-prerequisite way to play.

## What this package does

The `@mcptoolshop/star-freight` npm package is a thin launcher that:

1. Downloads the pre-built Star Freight binary from GitHub Releases
2. Verifies SHA256 checksums
3. Caches the binary locally
4. Runs it with your arguments

No Python, pip, or build tools needed. One command and you're playing.

## The game at a glance

You were a military pilot. Then you were a disgrace. Now you are a captain with a bad ship, no standing, and a star system that was already moving before you got here.

Five civilizations share a star system called the Threshold. Your crew determines your access. Your reputation determines your life. Every run becomes a different captain.
